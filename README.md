# Mapache Estudio

## Cómo correr el proyecto (Live Server)
1. Instala la extensión **Live Server** en VS Code.
2. Abre la carpeta del proyecto en VS Code.
3. Clic derecho en `index.html` → **Open with Live Server**.

> `fetch('./data/products.json')` requiere servidor (Live Server o GitHub Pages).

## Cómo publicar en GitHub Pages
1. Subí **todo el contenido** de esta carpeta (incluyendo `data/` y `assets/`).
2. En GitHub → Settings → Pages → **Source: main / root**.
3. Esperá ~2 min. Tu sitio quedará online.
