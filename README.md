# Mapache Estudio

## Cómo correr el proyecto
1. Instala la extensión **Live Server** en VS Code.
2. Abre la carpeta del proyecto en VS Code.
3. Haz clic derecho en `index.html` y selecciona **Open with Live Server**.

Esto es necesario porque el catálogo se carga con `fetch()` desde `products.json`.

## Funcionalidades
- Catálogo dinámico con fetch.
- Carrito en sidebar con LocalStorage.
- Toast al agregar producto.
- Login básico con validación.
- Formulario con Formspree.
